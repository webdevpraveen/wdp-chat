About This Anonymous Chat

Short Intro

This website is designed for completely anonymous chatting.

No accounts, no email, no phone number — nothing is required.

No personal data is collected at any point.

Chats are not stored, and every user connects with a random ID.

Privacy Policy

We do not store any chat messages on our server.

We do not collect any personal details, identity information, or device data.

Connections happen through temporary WebSocket sessions that end as soon as the user disconnects.

No tracking, analytics, cookies, or activity logging is used.

Terms & Usage Disclaimer

This platform is only a communication tool — the responsibility for misuse lies entirely with the user.

The website owner is not liable for any misuse, harm, or consequences caused by user behavior.

Note

This system is built to support open, simple, and barrier-free communication. Being anonymous does not give anyone the right to disrespect or harm others — please chat responsibly.
